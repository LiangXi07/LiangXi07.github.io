校园迷你马拉松记录

ZZX-LiangXi
校园活动

2025-10-11 08:38:21

人生中的第一件DIY跑衣，第一次正式参加的校园跑步比赛，6公里的"半马"挑战。从领取DIY体恤衫的兴奋，到起跑线上的紧张；从跑步App出问题的焦急，到坚持冲线时的释然——这场校园迷你马拉松，让我在奔跑中体验了坚持的力量，也收获了属于自己的完赛奖牌和难忘记忆。

🎽 赛前准备：拥有第一件DIY跑衣的兴奋

比赛前一周，学校给所有参赛选手发放了可以自由设计的白色跑衣。拿到那件纯白T恤时，我心里涌起一阵兴奋——这是我人生中第一件可以完全按照自己想法设计的运动衣。

穿上这件自己设计的跑衣站在起跑线上，感觉格外不同——它不是一件普通的运动服，而是专属于我的”战袍”，承载着我的期待和目标。

🏃 起跑时刻：技术小插曲与追赶

比赛当天清晨，校园里洋溢着兴奋的气氛。我和好友一起报名了6公里的”半马”项目，需要绕着学校指定的路线跑两圈。

🚦 起跑的意外：校园App的“小脾气”

站在起跑线前，我像往常一样打开校园跑App准备记录——结果发现因为没有及时开启权限，App完全无法使用。

那一刻真的有点慌：

  • 发令枪已响：周围的同学都开始奔跑
  • 手机还在手里：需要快速设置权限
  • 好友已经出发：看到朋友的身影逐渐远去
  • 边跑边操作：在移动中试图解决问题，步伐都乱了

大概花了2-3分钟，我终于调试好App开始正常记录。抬头一看，好友已经跑到了前面几十米的地方。

🏃‍♂️ 第一次追赶：不想掉链子的决心

看着朋友的背影，我心里只有一个念头：不能掉链子，要追上去！

  • 调整呼吸：深呼吸几次，找回跑步节奏
  • 加快步频：用比平时更快的频率迈步
  • 目光锁定：盯着前方那个熟悉的身影
  • 心里默念：“慢慢追，保持节奏，不能急”

🏁 赛中坚持：疲惫与毅力的较量

🔄 第一圈结束：疲惫感的突袭

我们学校的马拉松路线设计得很美——穿过林荫道、经过图书馆、绕过宿舍楼，最后回到起点完成一圈。但跑完第一圈（3公里）时，真实的疲惫感开始袭来。

身体的反应很诚实：

  • 腿部发酸：大腿和小腿肌肉开始发出抗议
  • 呼吸加重：需要更用力地呼吸才能保持供氧
  • 速度下降：不自觉地放慢了脚步
  • 思想动摇：“要不走一段吧”的念头开始出现

经过起点线时，我看到一些只报名3公里的同学已经完成比赛，正在领取完赛物资，心里不免有些羡慕。

💪 第二圈坚持：与自己的对话

第二圈是最考验意志力的阶段。每一公里都显得格外漫长，但我告诉自己：

“已经完成一半了，不能放弃”
“这件DIY跑衣不能白设计”
“说好要拿奖牌的，一定要坚持”

我鼓励着自己：

  • 配速提醒：“慢一点没关系，保持节奏”
  • 呼吸同步：尽量调整到相同的呼吸频率
  • 目标分解：“再跑500米就能看到图书馆了”
  • 互相打气：“加油，马上就剩最后一公里了”

🎉 冲线时刻：掌声与喝彩中的感动

🌟 最后冲刺：志愿者的力量

当看到终点线越来越近时，身体的疲惫似乎被一股新的力量取代。最让我感动的是冲线区域的志愿者们。

那一幕至今难忘：

  • 整齐的加油声：”加油！马上就到了！”
  • 热烈的掌声：为每一个冲线的选手鼓掌
  • 鼓励的眼神：志愿者们真诚的笑脸和眼神
  • 此起彼伏的喝彩：整个终点区域充满正能量

在欢呼声中，我冲过了终点线。那一刻，6公里的疲惫仿佛瞬间释放，取而代之的是巨大的成就感。

🏅 完赛喜悦：属于自己的奖牌

虽然最终成绩没有进入排行榜前列，但我得到了更珍贵的东西：

捧着那块完赛奖牌，我感到无比满足：

  • 沉甸甸的分量：不仅是金属的重量，更是坚持的重量
  • 纪念意义：这是我第一次参加校园马拉松正式跑步比赛的见证
  • 未来的起点：”希望这不是最后一块奖牌”

📊 跑步感悟：数字之外的收获

🌈 这场马拉松教给我的事

关于坚持：

  • 起步不顺不代表全程失败：App问题影响了开始，但不影响最终完赛
  • 疲惫时继续前进的每一步都有意义：第二圈的每一步都在锻炼意志力
  • 外界的鼓励能激发内在力量：志愿者的加油给了我最后冲刺的动力

关于友谊：

  • 好朋友是坚持的动力之一：知道有人在前面等我，让我必须追上去
  • 相互鼓励比独自奔跑更有力量：途中简单的”加油”就能带来能量
  • 共同的经历加深情感连接：一起流汗、一起坚持、一起冲线的记忆很珍贵

关于自我认知：

  • 我比想象中更能坚持：原以为自己会在第二圈放弃
  • 小障碍不影响大目标：技术问题只是小插曲，不影响完赛目标
  • 每一次尝试都在拓展边界：这次6公里，下次也许可以挑战更长距离

❤️ 赛后感想：奔跑中的青春印记

这场校园迷你马拉松，对我而言意义远不止一次体育活动。

那件DIY跑衣，从设计到完成，再到穿着它跑完全程，见证了我从准备到实现的完整过程。每次看到它，都会想起起跑前的期待和冲线后的喜悦。

那块完赛奖牌，虽然材质普通，但承载的是6公里的汗水、坚持和突破。它提醒我：只要不放弃，就能到达终点。

最珍贵的是那些瞬间：

  • 调试手机时的焦急
  • 追赶朋友时的决心
  • 第二圈时的自我对话
  • 冲线时听到的欢呼
  • 拿到奖牌时的笑容

这场奔跑让我明白：人生就像一场马拉松，起步可能有波折，途中会有疲惫，但只要保持前进的步伐，终能到达自己的终点。重要的不是速度多快、名次多高，而是那份坚持到底的勇气和冲线时刻的坦然。

感谢这次校园迷你马拉松，让我在奔跑中遇见了更坚韧的自己。我依然会继续在奔跑中，书写青春的篇章。

奔跑的意义，不在于超越多少人，而在于每一次都比昨天的自己前进了一步。
